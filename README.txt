YITH Purchased Column for WooCommerce allows reactivating the Purchased column removed in the last WooCommerce version (3.0.0 or greather)

== Description ==

Would you like to reactivate the Purchased column removed in the last WooCommerce version but you don't know how to do it? Easy! We have developed this plugin for you: after installing and activating it, you will have the column again just like before.
You can read more details [here](https://yithemes.com/blog/free-plugins/yith-purchased-column-for-woocommerce)

If you have problems with our plugins or something is not working as it should, first follow this preliminary steps:

== Installation ==

**Important**: First of all, you have to download and activate [WooCommerce](https://wordpress.org/plugins/woocommerce) plugin (version 3.0.0 or greather), because without it YITH Purchased Column for WooCommerce cannot work.

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH Purchased Column for WooCommerce` from Plugins page.

= Configuration =

YITH Purchased Column for WooCommerce will add a new tab called "Purchased Column" in "YITH Plugins" menu item. There, you will find all YITHemes plugins with quick access to plugin setting page.
